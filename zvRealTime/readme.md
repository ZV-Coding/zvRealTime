# zvRealTime

### Features
- Realtime
- Timezones: CET, GMT, ET, CT, MT, PT

#### Installation
1. Drag & Drop the Script of your server
2. Open your Server.cfg / CFG Editor in TxAdmin
3. Add "ensure zvRealTime" in your cfg
4. Go to server.lua
5. Choose the timezone you want to use for your server
6. Enjoy it!
